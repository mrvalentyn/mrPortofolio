My Story
behind this work

All of this began in 2020 during the COVID-19 pandemic. I was sitting 6+ hours on a computer per day, and I asked myself:"Do I sit there like a lazy playing some kind of games right? Does these games earn money by getting players on them right? Then I can do the same!"


I started learning Swift - a bad idea for begginers like me, a Windows user, but I would learn it again once more. I started then learning HTML - I thought it was hard, so I took a long pause till I got admitted into high school - a CS(Computer Science)-profiled class.
After some time being introduced to pseudocode and then, C++, I began writing randomly some lines of code while solving problems from different problem archives(pbinfo.ro & codeforces.com), constantly improving my algorithmic skills, understanding easier every piece of code from almost every programming language, C++ being non-begginer-friendly programming language.


And here I am, I have just learnt how to create some websites, soon profiling to desktop & mobile apps, and maybe games.
